# Workflow page

On this page you can assemble your workflow, by creating a list of tasks to be executed.

Each workflow is associated to a project, and the list of projects is [/projects](../projects.md).

After selecting a dataset, you can then run this workflow. Note that each dataset includes an image list, see <a href="https://fractal-analytics-platform.github.io/image_list/" target="_blank">https://fractal-analytics-platform.github.io/image_list</a>.

After running a workflow, you will also get information about the outcome of each task.
